Assignment 1: HTML5 Tags and Attributes

This document demonstrates a comprehensive understanding of HTML5 tags and their associated attributes.

Article Element

The <article> tag is used for independent, self-contained content, like a blog post or news story.

Aside Element

The <aside> tag is for content that is related to the main content but separate from it, like a sidebar.


    
Related Information

    
This information is in a sidebar.

Audio Element

The <audio> tag embeds sound content. The controls attribute adds play, pause, and volume controls.

Button Element

The <button> tag creates a button. The type attribute specifies the button's type.

Canvas Element

The <canvas> tag is a container for graphics. It uses attributes like width and height.


    Your browser does not support the canvas element.

Datalist Element

The <datalist> tag provides a list of options for an <input> field. The list attribute of the input is linked to the datalist's id.

Details and Summary Elements

The <details> tag creates an expandable section. The <summary> tag provides the visible heading for the details section.


    Click to view more
    
Here is the extra content that is hidden by default.

Dialog Element

The <dialog> tag creates a popup dialog box. The open attribute makes the dialog visible.

This is a dialog box.

Embed Element

The <embed> tag is for embedding external content like a PDF or a video. It uses the src and type attributes.

Figure and Figcaption Elements

The <figure> tag represents a self-contained content block, often with a caption provided by the <figcaption> tag.

Footer Element

The <footer> tag contains information about the author, copyright, or contact information.


    
© 2025 All rights reserved.

Header Element

The <header> tag represents introductory content, often with navigation or a heading.


    
Welcome to My Website

Main Element

The <main> tag contains the primary content of the document.


    
This is the main content of the page.

Mark Element

The <mark> tag is used to highlight parts of text for emphasis.

This is some text with a highlighted word.

Meter Element

The <meter> tag represents a measurement within a given range, like a fuel gauge. Attributes include value, min, max, low, high, and optimum.

Usage: 60%

Nav Element

The <nav> tag contains navigation links to other pages or sections.


    Home |
    About |
    Contact

Progress Element

The <progress> tag represents a progress bar for a task. The value and max attributes specify the completion status.

Section Element

The <section> tag groups related content. It is a thematic grouping of content, typically with a heading.


    
Section Title

    
This is a paragraph within a section.

Time Element

The <time> tag defines a date or time. The datetime attribute provides a machine-readable format.

The event will be held on October 26th at 8 PM.

Video Element

The <video> tag embeds a video. The controls attribute adds the playback controls, and the poster attribute specifies an image to show while the video is downloading.

WBR Element

The <wbr> tag provides a line break opportunity for a long word or URL to prevent overflow.

Template Element

The <template> tag holds content that will not be displayed immediately. It is useful for creating reusable HTML fragments with JavaScript.
